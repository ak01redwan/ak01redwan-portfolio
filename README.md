# Professional Developer Portfolio

A world-class, high-performance personal portfolio website built with modern web technologies. This project showcases my skills, experience, and projects in a clean, interactive, and accessible way.

## 🚀 Features

- **Multilingual Support (i18n)**: Fully supports English and Arabic with RTL (Right-to-Left) layout optimization.
- **Dark/Light Mode**: Seamless theme switching with system preference auto-detection.
- **Responsive Design**: Optimized for all screen sizes, from mobile to ultra-wide desktops.
- **Performance Optimized**: 
  - Lazy loading of pages and components.
  - Image optimization using modern formats.
  - Minimal bundle size with Vite.
- **Interactive UI**: Smooth animations powered by `motion` (Framer Motion).
- **SEO Ready**: Dynamic meta tags and titles for every page.
- **Accessible**: Semantic HTML, ARIA labels, and keyboard navigation support.
- **Data Driven**: Centralized data management for easy updates.

## 🛠️ Tech Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS
- **Animations**: Framer Motion
- **Internationalization**: i18next
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Deployment**: Optimized for Vercel / Netlify / Cloud Run

## 📂 Project Structure

```text
src/
├── components/     # Reusable UI components
├── data/           # Centralized static data (projects, skills, etc.)
├── locales/        # Translation files (EN, AR)
├── pages/          # Page-level components
├── App.tsx         # Main application entry and routing
└── main.tsx        # React mounting point
```

## 🚦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
